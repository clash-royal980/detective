  1. 信息学部

    1. 信息技术学部下设计算机科学与技术系、电子信息工程与自动化系，以本科教育为主，专科教育为辅。学部现有教师37人，90%以上教师具有硕士以上学历，其中包括教授2人，副教授7人，高级工程师1人。
    2. 现代社会的各个领域及人们日常生活等都与信息技术有着紧密的联系。能综合运用所学知识和技能，适应现代电子技术发展的要求，从事企事业单位与相关的产品及设备的生产、安装调试、运行维护、销售及售后服务、新产品技术开发等应用型技术人才和管理人才是社会发展和经济建设的客观需要，市场对该类人才的需求越来越大。本学部所属专业，无论在就业岗位，工薪待遇，还是发展前景上，具有较大的优势。学部不断深化教学改革，大力开展第二课堂活动，夯实基础技能与操作，注重岗位技能和职业道德的培养，建立和完善教学质量监控系统，教学质量不断得到提高。学部专业设置、人才培养方案与社会经济发展紧密结合并动态调整，提出并实行了分类培养、订单教育的办学模式，建立了广泛的校企合作的教学实训基地。无论你将来从事信息技术领域软件设计，还是硬件开发，你都将受到科学的专业训练。

  2. 管理学部

    1. 管理学部下设财务管理系、市场营销系和旅游管理系，以本科教育为主，专科教育为辅。已开设财务管理、市场营销、旅游管理本科专业和市场营销、旅游管理、会计电算化、会计电算化（涉外）专科专业。
    2. 学部现有自有专任教师27人，其中教授1人，副教授3人，讲师14人；外聘专任教师23人，其中教授1人、副教授14人、讲师6人，形成了一支相对稳定、结构合理、学术水平高、教学经验丰富、责任心强的育人团队，有效地保证了教育教学质量。
    3. 学部坚持“笃学力行、立德励志”的育人理念和“崇尚科学、探求真理、严谨治学、敬业创新、与时俱进、团结拼搏”的学部建设理念；倡导“善学、善问、善待问”的进学之道。在教学中，坚持理论与实际、传授知识与发展智能、统一要求与因材施教、发挥教师的主导作用与学生学习的能动作用相结合的原则，使培养的学生身心健康、有责任感、对社会需求的适应性强、就业面广、发展潜力大、富有创新和创业精神，并注重发展学生的个性。 学部学生活动丰富多彩。每年上半年举办“学生文化月”活动，目前已举办七届，包括书画大赛、主持人大赛、演讲比赛等充分锻炼了学生各种能力，展示了学生个性。
    4. 学部的学生在参加全国、省、市组织的活动中及历年考研中均取得了优异的成绩，考取研究生的学校有武汉大学、华中科技大学、华中师范大学、中南财经政法大学等名校。

  3. 经济学部

    1. 江汉大学文理学院经济学部在原经济管理系(2002年成立)的基础上于2013年9月组建，致力于培养高质量的经济类复合型、应用型高级专门人才。
    2. 学部以本科教育为主。立足国内，面向国际，以经济建设和社会发展对人才的需求来设置专业、确定培养目标。已开设国际经济与贸易、金融学两个本科专业，其中金融学专业获得省教育厅首批“湖北省独立学院重点培育专业”。现有在校生1398人。
    3. 学部现有专职教师22人，其中教授1人，副教授3人，讲师8人。特聘教授12人，副教授38人；依托江汉大学并充分利用武汉地区高校教师资源优势，形成了一支相对稳定、结构合理、学术水平高、教学经验丰富、责任心强的创业团队，有效地保证了教育教学质量。
    4. 学部在教学中，坚持理论与实际、传授知识与发展智能、统一要求与因材施教、发挥教师的主导作用与学生学习的能动作用相结合的原则，使培养的学生身心健康、对社会需求的适应性强、就业面广、发展潜力大、富有创新和创业精神，并注重全面发展学生的个性。学生在参加全国、省、市组织的活动中及历年考研中均取得了优异的成绩，考取学校有武汉大学、华中师范大学、中南财经政法大学等名校。

  4. 外语学部

    1. 江汉大学文理学院外语学部是学院重点建设学部之一，下设英语系、法语系和大学外语系。创办近10年来，随着学院的发展和壮大，外语学部在专业设置、课程改革、师资队伍建设和教学科研水平提高等方面都取得了长足的进步，办学规模不断扩大，现有在校本科生、专科生共1032人。现已开设英语专业（本科），下设英语语言文学、英语翻译、师范英语、英法双语等四个模块；商务英语（本科）；英语专业（专科），下设旅游英语和商务英语等两个专业；法语专业（本科），现已招两届学生，共计56人。大学外语系承担全院大学英语教学和英语、法语两系的二外教学任务。
    2. 外语学部现有的自有教师、教辅老师和外教41名，并聘请武汉大学、江汉大学及其他院校教学经验丰富、教学认真负责的教授和副教授任教。专任教师58人，其中教授4人，副教授 22人，讲师26人，助教6人；教师中有硕士学历学位27人，正在攻读硕士学位5人。专业必修课主要由具有硕士学历学位和讲师以上职称的教师主讲，高级专业技术职务教师给本科生授课率为100%。
    3. 本专业学生的英语基础知识与技能均比较扎实、实践创新能力较强，综合素质较高。学生参加2010年的全国研究生入学考试，初试上线31人。学生参加全国大学生英语竞赛、全省英语翻译大赛、CCTV（现改名为外研社）杯湖北赛区英语演讲大赛决赛中均有上乘表现：如英语专业06级专科生邝立（三等奖）、07本科生级杨梅（二等奖），09级学生本科生徐义伟（三等奖）、张颖璐（二等奖）在CCTV杯湖北赛区英语演讲比赛决赛中获得二、三等奖，特别是杨梅同学在全省各高校42名决赛选手中取得第五名的好成绩，技压众多重点院校的选手。本专业毕业生就业率在文理学院一直名列前三名，2011届毕业生就业率排名第一。考研率逐年上升，2011届毕业生考研率达10%。
    4. 外语学部有21间语言实验室，其中11间为江汉大学共享资源、1个语言情景实验室、1座英语专业自主学习中心、1个英语报纸编辑部和开展第二活动的“文理之声”调频电台。学部为了使学生适应未来实习、就业的需要，建立了校内大学外语教学实习基地，校外建立了10多个学生实习基地或实习点，行业涉及中小学校、旅游公司、涉外酒店、合资企业。在课程设置上，学部充分考虑学生今后就业的需要，开设一些旨在提高学生人文素质和谋生技巧的科目。另外，学部学生活动丰富多彩，如英语节、英语角、演讲比赛、小品比赛、外语歌曲比赛等，使学生在参与各项活动中提高了相互沟通能力、语言表达能力、社会交往能力和活动组织能力。
    5. 几年来，学部青年教师积极参加全国、全省英语教学研讨会，进行大会专题发言，提供教学示范观摩课，受到专家同行们的一致好评。在全省、市级教师教学比武中不少教师分获二、三等奖、教学五项全能三等奖。
